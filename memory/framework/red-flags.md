# 风险信号库

> 发现任一信号必须在结论中明确讨论。
> 不意味"一票否决"，但必须解释为什么在此案例中不致命。

## 财务危险信号
- [ ] 净资产为负
- [ ] OCF 连续3年为负
- [ ] 应收账款增速持续 > 营收增速
- [ ] 存货周转天数持续上升
- [ ] 负债率 > 80% 且连续亏损
- [ ] 商誉/总资产 > 30%
- [ ] 关联交易占比异常
- [ ] 分红率 > 100% 或借钱分红

## 业务危险信号
- [ ] 核心产品/技术被替代（结构性死亡，非周期）
- [ ] 大客户依赖 > 50%
- [ ] 管理层大量减持
- [ ] 审计师频繁更换
- [ ] 被监管调查/做空狙击

## 估值危险信号
- [ ] 乐观假设下估值也不合理
- [ ] 市场共识过于乐观
